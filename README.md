# My Travel Planner

### 👨‍🏫프로젝트 소개

My Travel Planner은 사용자가 지도와 다양한 추천 패널을 통해 여행 명소를 추천받고, 
그 정보를 바탕으로 여행 계획을 세울 수 있는 풀스택 웹 애플리케이션입니다.
여행을 다녀온 후에는 여행 일지를 작성할 수 있는 기능도 제공하고 있습니다.
